Memory Arena Allocator

Coded in C, this is a memory allocation program that can implement different memory management styles using linked lists. Some of the memory allocation algorithms include best fit, worst fit, first fit, and next fit, with the fastest of the four algorithms only taking 1.84 milliseconds longer than a call to malloc on average. This program can also free blocks of memory and merge adjacent free memory blocks accordingly.

Tools used: C, GDB, GCC
